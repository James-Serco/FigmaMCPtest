# Real Property Hub

A React application implementing a dashboard for managing real property transactions, built with Vite and TypeScript.

## Features

- Transaction Plans Management
- Interactive Charts (Bar Charts and Doughnut Charts)
- Search Functionality
- Responsive Design with Tailwind CSS

## Getting Started

### Prerequisites

- Node.js (version 18 or higher)
- npm or yarn

### Installation

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open your browser and navigate to `http://localhost:5173`

## Build

To build the project for production:

```bash
npm run build
```
